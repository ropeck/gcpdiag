dataproc/BP/2022_001

Cloud Monitoring agent is enabled.

Product: Cloud Dataproc
Rule class: BP - Best practice, opinionated recommendation

Description

Memory and disk usage metrics are often useful when troubleshooting, however, the Cloud Monitoring agent is not enabled by default when a cluster is created.

Remediation

Enable dataproc.monitoring.stackdriver.enable during cluter creation.

Further information

  • Cloud Monitoring cluster metrics
